FlightOps

Turnaround Management System

Purpose

FlightOps is designed to provide a single operational view of a flight turnaround. It combines operations, passenger handling, ramp services, SPUR activities, special handling, assignments, flight events, operational chat and readiness tracking in one place.

Main Features

  • Real-time flight turnaround monitoring
  • Department-based permissions
  • Flight assignments
  • Special handling management
  • Operational chat
  • Audit trail
  • Automatic flight status updates
  • Readiness calculation
  • Manual flight creation
  • Live updates using Socket.IO

Technology

  • Node.js
  • Express
  • MongoDB Atlas
  • Socket.IO
  • Bootstrap 5
